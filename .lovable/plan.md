
# The Aqua Retreat — Premium Real Estate Landing Page

## Overview
A stunning, single-page real estate landing page for "The Aqua Retreat at Godrej Park World, Hinjawadi Phase 1, Pune" with an aqua/water-inspired design language. Visual polish and smooth animations are the priority. Contact form redirects to WhatsApp. Placeholder images used throughout, ready for swapping.

## Design System
- **Primary:** Serene Aqua Blue (#5EBCD6)
- **Secondary:** Deep Ocean Blue (#1B5E7F)
- **Accent:** Sandy Gold (#D4AF37)
- **Neutrals:** Cool grays and white
- **Typography:** Clean, premium font pairing (Inter/Playfair Display style via system fonts)
- **Effects:** Glass-morphism, water gradients, ripple hover effects, smooth scroll animations
